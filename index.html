Project on Console

Visual Studio Code is a sorce-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refacoring, and enbedded Git.

The console. log() is a function that writes a message to log on the debugging console, such as Webkit or firebug. In a browser you will not see anything on the screen. It is only available in Firefox with firebug and in Webkit based browsers (Chrome and safari)
